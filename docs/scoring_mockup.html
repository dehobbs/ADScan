Certificate template "UserAuth" allows the enrollee to specify an arbitrary Subject Alternative Name (SAN). Combined with domain authentication, this allows any authenticated user to obtain a certificate for any account including Domain Admins.

The Web Enrollment (certsrv) endpoint is accessible without HTTPS and does not enforce Extended Protection for Authentication (EPA), enabling NTLM relay attacks against the CA.
